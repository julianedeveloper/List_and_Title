Tags em HTML

Em HTML 5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizara em cada caso.

  • Páragrafos
  • Título
  • Inserir um link
  • Inserir uma imagem
  • Inserir um rodapé
  • Listas com ordem numerada
  • Lista sem ordem, somente marcadores.
  • Inserir um menu
